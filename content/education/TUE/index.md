## Projects

### ASML (12 months individual project)

* To conduct a comparative benchmark of multiple data‑streaming technologies to evaluate throughput, resource usages, and reliability.
* To design a high‑throughput data‑streaming interface based on benchmark insights and system constraints
* To prototype and implement the new interface within the Twinscan system to enable reliable large‑volume data transfer

---

### Canon (2 months in-house team project)

* Led the exploration and implementation of Modelix Editor.
* Enabled the existing prototyping application to function as a web‑based modeling tool.
* Documented key findings and development guidelines to support future successors.

### Differ (2 months in-house team project)

* Designed system architecture and UI data flow for a drag‑and‑drop SDL simulation builder
* Delivered PoC demo enabling configurable electrochemistry simulations with modular frontend logic

### TU/e Building Acoustics (2 months in-house team project)

* Delivered auralization support and streamlined data handling to improve simulation usability
